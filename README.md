# The Movie

React + Next + TypeScript + Antd + YTS API 를 이용한 영화 검색 웹 사이트 입니다.

## 기능

- [ ] 영화 리스트
- [ ] 영화 정렬
- [ ] 영화 필터
- [ ] 영화 상세보기
